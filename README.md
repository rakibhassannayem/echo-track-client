# EcoTrack

Live site (client): https://ecotrack-dd506.web.app

EcoTrack is a lightweight React + Vite client for tracking personal and community sustainability challenges. It connects to the Echo Track API to browse challenges, join and track progress, create community challenges, and share tips & events.

Features
- Join and track progress on sustainability challenges (daily progress, completion tracking).
- Create, edit and manage challenges you own (admin-style UI for creators).
- Community tips and upcoming events to discover and share best practices.
- Firebase authentication (email / Google) to keep user activity private and synced.
- Responsive, accessible UI built with Tailwind + DaisyUI for fast development and pleasant UX.
